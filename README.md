# Империя Востока 3D RTS

Готовый браузерный проект без сборщика. Подходит для GitHub Pages и других статических хостингов.

## Что внутри
- реальное время вместо кнопки "Ход"
- 3D карта на Three.js
- hex-рельеф с высотой, реками, лесом, холмами и скалами
- строительство по времени
- непрерывная экономика в секунду
- обучение юнитов
- вражеские лагеря и набеги
- башни и гарнизоны автоматически защищают империю
- исследования
- localStorage сохранение
- подключение `.glb` моделей из `assets/models/buildings/`

## Структура
- `index.html` — точка входа
- `styles.css` — весь UI
- `src/` — модульная игровая логика
- `assets/models/buildings/` — твои модели зданий

## Какие модели нужны
Положи сюда файлы:

- `town-center.glb`
- `farm.glb`
- `storage-shed.glb`
- `mine.glb`
- `market-stalls.glb`
- `barracks.glb`
- `stone-wall.glb`
- `temple.glb`
- `storage-house.glb`
- `port.glb`
- `academy.glb`
- `wonder.glb`
- `fortress.glb` (для башни)

Если части моделей нет, игра покажет fallback-примитив и не сломается.

## Как запустить локально
Просто открой папку через локальный сервер. Например:

```bash
python3 -m http.server 8080
```

Потом открой `http://localhost:8080`

## Как залить на GitHub Pages
1. Загрузи весь проект в репозиторий
2. В Settings → Pages выбери ветку `main` и корень `/root`
3. Убедись, что `.glb` доступны по прямым URL

## Управление
- свайп / мышь — вращение камеры
- тап / клик по клетке — выбор
- `Строить` — выбрать постройку, затем коснуться клетки
- `Войска` — обучение юнитов в столице и казармах
- `Знания` — исследования
- `⏸ / x1 / x2 / x4` — скорость симуляции

## Важно
Проект уже адаптирован именно под веб-игру:
- без сборщика
- без серверной части
- без тяжёлого backend
- рассчитан на дальнейшее расширение прямо в браузере
